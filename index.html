Entorno Ubuntu 20.04 de propósito general

El entorno Ubuntu es un entorno flexible de un solo nodo. La conexión al entorno se establece como usuario root. Esto brinda total flexibilidad para instalar paquetes adicionales, explorar las funciones internas de Linux y experimentar con nuevas ideas.

Además, el entorno viene con las siguientes herramientas ya instaladas:

  • Docker
  • Git

Recordá que el entorno es efímero. Es decir, cualquier cambio realizado en el sistema se perderá una vez perdida la conexión.
